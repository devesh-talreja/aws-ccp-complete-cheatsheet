## CLOUD CONCEPTS (FUNDAMENTAL)

### Six Advantages of Cloud Computing:
1. **Trade capital expense for variable expense:** Pay only what you use
2. **Benefit from massive economies of scale:** Lower prices
3. **Stop guessing capacity:** Scale up/down as needed
4. **Increase speed and agility:** Resources in minutes
5. **Stop spending money on data centers:** Focus on business
6. **Go global in minutes:** Deploy worldwide quickly

### Cloud Deployment Models:
1. **Public Cloud:** AWS, Azure, GCP (fully on cloud)
2. **Private Cloud:** On-premises (you manage everything)
3. **Hybrid Cloud:** Mix of public and private (best of both)

### Shared Responsibility Model:
**AWS Responsible for:** Security OF the cloud (hardware, software, facilities)
**You Responsible for:** Security IN the cloud (data, IAM, encryption, OS patching for EC2)

**Memory Trick:** 
- AWS = Building security (locks, guards, structure)
- You = What you put in the building (your valuables, who you let in)

### Types of Cloud Computing:
1. **IaaS (Infrastructure):** EC2, VPC (most control) - "You manage OS and above"
2. **PaaS (Platform):** Elastic Beanstalk, RDS - "You manage just the application"
3. **SaaS (Software):** Gmail, Salesforce - "You just use the application"
