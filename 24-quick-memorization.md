## QUICK MEMORIZATION TECHNIQUES

### Memory Palaces for Services:
**Storage Room:** S3 (boxes), EBS (hard drives on shelf), EFS (shared filing cabinet)
**Security Office:** IAM (ID badges), KMS (key locker), WAF (bouncer at door)
**Computer Lab:** EC2 (desktops), Lambda (laptops that appear when needed)
**Library:** RDS (organized bookshelves), DynamoDB (messy but fast desk)

### Songs/Rhymes:
- "S3 is the key, for object storage you see"
- "CloudWatch will watch, CloudTrail will track, Config shows what changed, and that's a fact"
- "EC2 is virtual, Lambda is light, both compute resources, working day and night"

### Mnemonics:
- **VPC Components:** "Sir, NAT Route Into Secure Networks" = Subnets, NAT, Route Tables, IGW, Security Groups, NACLs
- **RDS Engines:** "My Poor Older Sister Ate Pickles" = MySQL, PostgreSQL, Oracle, SQL Server, Aurora, (MariaDB)
