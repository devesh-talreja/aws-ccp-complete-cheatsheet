## 8. ANALYTICS

### Athena
**What:** Query S3 data using SQL
**Real-life:** Like using SQL to search through files in a folder
**Key Points:**
- Serverless
- Pay per query
- Standard SQL
**Memory Trick:** "Athena = Ancient wisdom = Query knowledge in S3"

### EMR (Elastic MapReduce)
**What:** Big data processing using Hadoop, Spark
**Real-life:** Like a massive parallel processing system for huge datasets
**Memory Trick:** "EMR = Emergency big data processing"

### Kinesis
**What:** Real-time data streaming
**Real-life:** Like a live video stream of data
**Key Services:**
- **Kinesis Data Streams:** Capture streaming data
- **Kinesis Firehose:** Load data into AWS stores
- **Kinesis Analytics:** Analyze streams with SQL
**Memory Trick:** "Kinesis = Kinetic = Moving data in real-time"

### Glue
**What:** ETL (Extract, Transform, Load) service
**Real-life:** Like a data preparation service that cleans and transforms data
**Key Points:**
- Serverless
- Data catalog
**Memory Trick:** "Glue = Glues data together = ETL"

### QuickSight
**What:** Business intelligence and visualization
**Real-life:** Like Tableau or Power BI
**Key Points:**
- Serverless
- Create dashboards
**Memory Trick:** "QuickSight = Quick insights = Visualization"
