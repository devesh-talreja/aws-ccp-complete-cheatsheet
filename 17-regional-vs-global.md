## REGIONAL SERVICES VS GLOBAL SERVICES

### Global Services (Not region-specific):
- **IAM:** Users, Groups, Roles, Policies
- **Route 53:** DNS service
- **CloudFront:** CDN
- **WAF:** Web Application Firewall
- **Organizations:** Multi-account management

### Regional Services (Must choose region):
- **EC2:** Virtual machines
- **RDS:** Databases
- **VPC:** Networking
- **S3:** Object storage (BUT bucket names are globally unique)
- **Lambda:** Serverless compute

**Memory Trick:** "Identity, DNS, and CDN are global - everything else is regional"
