## CATEGORIZED SUMMARY

### COMPUTE
**Managed:** Lambda, Fargate, Elastic Beanstalk
**Unmanaged:** EC2
**Containers:** ECS, EKS, ECR
**Simple:** Lightsail
**Batch Processing:** Batch

### STORAGE
**Object:** S3 (+ Glacier)
**Block:** EBS
**File:** EFS, FSx
**Backup:** AWS Backup
**Transfer:** Snow Family, DataSync, Transfer Family, Storage Gateway

### DATABASE
**Relational:** RDS, Aurora
**NoSQL:** DynamoDB
**Cache:** ElastiCache (Redis/Memcached)
**Data Warehouse:** Redshift
**Graph:** Neptune
**Time-series:** Timestream
**Ledger:** QLDB

### NETWORKING
**Foundation:** VPC (Subnets, IGW, NAT, Route Tables)
**Security:** Security Groups, NACLs, WAF, Shield
**Load Balancing:** ALB, NLB, Gateway LB
**DNS:** Route 53
**CDN:** CloudFront
**Connectivity:** Direct Connect, VPN, PrivateLink
**API:** API Gateway

### SECURITY & COMPLIANCE
**Identity:** IAM, Cognito, Organizations, SSO
**Encryption:** KMS, CloudHSM, Certificate Manager
**Secrets:** Secrets Manager, Parameter Store
**Compliance:** Artifact, Audit Manager
**Threat Detection:** GuardDuty, Inspector, Detective, Macie
**DDoS Protection:** Shield

### MANAGEMENT
**Monitoring:** CloudWatch (metrics, logs, alarms)
**Audit:** CloudTrail, Config
**Automation:** Systems Manager, OpsWorks
**Infrastructure as Code:** CloudFormation, CDK
**Best Practices:** Trusted Advisor
**Governance:** Control Tower, Service Catalog

### APPLICATION INTEGRATION
**Queuing:** SQS
**Notifications:** SNS
**Events:** EventBridge
**Workflow:** Step Functions
**API:** API Gateway, AppSync

### ANALYTICS
**Query:** Athena
**Big Data:** EMR, Kinesis
**ETL:** Glue
**Visualization:** QuickSight
**Data Lake:** Lake Formation

### MACHINE LEARNING
**General:** SageMaker
**Vision:** Rekognition
**Language:** Comprehend, Translate, Transcribe, Polly
**Chatbots:** Lex
**Forecasting:** Forecast
**Recommendations:** Personalize
**Fraud Detection:** Fraud Detector

### MIGRATION
**Planning:** Application Discovery Service, Migration Hub
**Database:** DMS (Database Migration Service)
**Server:** Application Migration Service (MGN)
**Data Transfer:** Snow Family, DataSync, Transfer Family, Storage Gateway

### COST MANAGEMENT
**Monitoring:** Cost Explorer, Cost and Usage Report
**Budgeting:** AWS Budgets
**Planning:** Pricing Calculator
**Optimization:** Compute Optimizer, Trusted Advisor
**Savings:** Reserved Instances, Savings Plans, Spot Instances

### DEVELOPER TOOLS
**CI/CD:** CodeCommit, CodeBuild, CodeDeploy, CodePipeline
**IDE:** Cloud9
**Application Testing:** Device Farm
**API Management:** API Gateway

### END USER COMPUTING
**Virtual Desktops:** WorkSpaces
**Application Streaming:** AppStream 2.0
**File Sharing:** WorkDocs
