## 12. OTHER IMPORTANT SERVICES

### CloudFormation
**What:** Infrastructure as Code (IaC)
**Real-life:** Like a blueprint that automatically builds your infrastructure
**Key Points:**
- Templates in JSON or YAML
- Version control infrastructure
- Repeatable deployments
**Memory Trick:** "CloudFormation = Cloud formation = Build infrastructure from templates"

### Elastic Load Balancing (ELB)
**What:** Distribute traffic across multiple targets
**Three Types:**
1. **Application Load Balancer (ALB):** Layer 7 (HTTP/HTTPS), advanced routing
2. **Network Load Balancer (NLB):** Layer 4 (TCP/UDP), ultra-high performance
3. **Gateway Load Balancer:** Layer 3, for virtual appliances
**Memory Trick:** "Load Balancer = Balance the load = Distribute traffic"

### Auto Scaling
**What:** Automatically adjust capacity
**Real-life:** Like automatically hiring temporary staff during busy hours
**Key Points:**
- Maintain application availability
- Scale EC2, DynamoDB, Aurora, ECS
**Memory Trick:** "Auto Scaling = Automatic sizing = Adjust capacity"

### Outposts
**What:** Run AWS infrastructure on-premises
**Real-life:** Like having AWS servers in your own data center
**Key Points:**
- Hybrid cloud
- Low latency, local data processing
**Memory Trick:** "Outposts = Outpost in your location = AWS on-premises"

### WorkSpaces
**What:** Virtual desktop infrastructure (VDI)
**Real-life:** Like accessing your work computer from anywhere
**Memory Trick:** "WorkSpaces = Work from anywhere = Virtual desktop"

### AppStream 2.0
**What:** Stream desktop applications
**Real-life:** Like running Photoshop in a browser
**Memory Trick:** "AppStream = Stream applications = Application streaming"

### IoT Core
**What:** Connect IoT devices to AWS
**Real-life:** Like managing smart home devices
**Memory Trick:** "IoT Core = Internet of Things = Connect devices"

### Elastic Transcoder / MediaConvert
**What:** Convert media files to different formats
**Real-life:** Like converting a video for different devices
**Memory Trick:** "Transcoder = Transcode media = Convert formats"
